# 🚀 语音助手应用中心

这是一个基于 Web 的语音助手应用中心，提供插件扩展和云端存储功能的一站式服务平台。

## 📋 项目简介

本项目是一个语音助手的应用中心网页，包含两个核心功能模块：

- **🎨 插件市场**：自定义语音助手的界面风格和功能扩展
- **☁️ GitHub 云盘**：安全可靠的云端文件存储服务

## ✨ 主要功能

### 1. 插件市场 (`plugins.html`)
- UI 美化插件 - 自定义配色方案
- 多种主题风格一键切换
- 布局样式随心调整
- 持续更新更多功能插件

### 2. GitHub 云盘 (`cloud.html`)
- 免费 1GB 存储空间
- 支持文件上传/下载/删除
- 跨设备实时同步
- 文件拖拽便捷操作

## 🛠️ 技术栈

- **前端**：HTML5 + CSS3 + JavaScript
- **样式**：响应式设计，支持移动端和桌面端
- **后端集成**：GitHub API（云盘功能）

## 📁 文件结构

```
server/
├── index.html # 主页面（应用中心入口）
├── plugins.html # 插件市场页面
└── cloud.html # GitHub 云盘页面
```
## 🚀 使用方法

### 本地访问
直接双击 `index.html` 文件即可在浏览器中打开应用中心。

### 在线访问
可以将此仓库部署到 GitHub Pages，获得在线访问链接。

## 💡 特性亮点

- ✅ **美观的 UI 设计**：渐变色彩、卡片式布局、流畅动画
- ✅ **响应式设计**：完美适配各种屏幕尺寸
- ✅ **离线使用**：支持离线访问，数据保存在本地或 GitHub
- ✅ **跨平台**：支持 Windows、macOS、Linux 等所有主流操作系统
